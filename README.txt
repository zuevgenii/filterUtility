Инструкция по запуску программы фильтрации данных в файле
Версия JAVA: java version "18.0.1" 2022-04-19
             Java(TM) SE Runtime Environment (build 18.0.1+10-24)
             Java HotSpot(TM) 64-Bit Server VM (build 18.0.1+10-24, mixed mode, sharing)
Система сборки (MAVEN) : Apache Maven 3.9.9
Зависимости: Нет внешних зависимостей. Программа использует только стандартные библиотеки Java
             (java.io, java.util, java.util.logging). Это делает её максимально простой в развертывании.
Запуск программы: для запуска программы необходимо выполнить команду
             java -jar filterUtility-1.0-jar-with-dependencies.jar [входные_файлы] [опции]
Опции командной строки:
             -o <каталог>: Указывает каталог для выходных файлов. Если не указано, выходные файлы создаются
              в текущей директории. Пример: -o output_dir
             -a: Включает режим добавления в существующие выходные файлы. Если не указано, выходные файлы
              перезаписываются.
             -p <префикс>: Задает префикс для имен выходных файлов. Пример: -p myprefix_
              Выходные файлы будут называться myprefix_integers.txt, myprefix_doubles.txt и
              myprefix_strings.txt.
              -s: Включает вывод краткой статистики (только количество элементов).
              -f: Включает вывод полной статистики (количество, минимум, максимум, сумма, среднее для чисел;
               количество, минимальная длина, максимальная длина для строк).
Логирование:   Все ошибки и предупреждения записываются в файл FileContentFilter.log, который создается в
               директории, из которой запускается программа. Рекомендуется проверять этот файл при возникновении
               проблем.
Обработка ошибок:  Программа старается обрабатывать все возможные ошибки и не “падает”. В случае
               возникновения ошибок, в консоль выводятся информативные сообщения, а подробности записываются
               в лог-файл. При возникновении критических ошибок, продолжение работы может быть невозможно,
               и программа завершается.
